# tkinter User Graphic Interface 使用者介面
This program made with Python's popular GUI module Tkinter to plot the graphics you need.

## Installation
```Python
$ pip install tk
$ conda install tk
```

## Run the script
```Python
$ pyhton GUI.py
```
